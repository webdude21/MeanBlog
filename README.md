MeanBlog
========
### Blog system using the M.E.A.N stack

### The Application can be found live here(http://meanestblog.herokuapp.com/)

Used technologies
------------

1. node.js
2. Express
3. MongoDb
4. Mongoose
5. Angularjs


There are three user roles in the application

1. Administrator
2. Emplpoyee
3. Customer
